# core_rooibos
Directory containing the HW and software for Core Rooibos; A Fully Adiabatic, Reversible, and SuperScalar (FARS) extension of the OpenTitan Earlgrey Core

This repo is a work in progress and as such, it is not guaranteed that the core will immediately run for you when inserted into a local copy of the OpenTitan directory.

Best Known Method (BKM) for running in opentitan:

cp -fR hw/top_earlgrey /path/to/opentitan/hw/

Run using the Bazel steps given in the opentitan documentation.
